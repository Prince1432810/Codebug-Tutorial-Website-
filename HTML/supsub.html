Superscript in HTML :

Superscript is a typographic style where text or characters are positioned above the baseline, appearing smaller and slightly elevated. In HTML, the <sup> element is used to denote and render superscript text. It is commonly employed for various purposes, including mathematical notations, footnotes, and annotations.


HTML <sup> Element:

The <sup> element is an inline element that defines superscript text within an HTML document. When this element is used, the text inside it is displayed in a smaller size and positioned above the normal text baseline.

Syntax : 
<p>This is an example<sup>1</sup> of superscript.</p>

Example Use Cases :

  1. Mathematical Notations :
    This would render the Pythagorean theorem: a² + b² = c².
    2. This would render the Pythagorean theorem: a² + b² = c².
  2. Chemical Formulas :
    This represents a water molecule with an additional proton
    3. <p>H<sup>2</sup>Osup>2<+sup/>2</p>
  3. Footnotes :
    At the bottom of the document or article, you can include the corresponding footnote:
    4. <sup> 1</sup> Additional information about the topic.
    At the bottom of the document or article, you can include the corresponding footnote: 
    <sup>1</sup> Additional information about the topic.

Styling with CSS:

While the <sup> element defines the structure, you can use CSS to style the superscript text further. For example:

sup {
  font-size: smaller;
  vertical-align: super;
  /* Additional styling properties */
}

This CSS rule sets the size of the superscript text to be smaller than the surrounding text and vertically aligns it to appear above the baseline.

HTML Entities for Superscript:

In addition to the <sup> element, you can use HTML entities to represent common superscript characters. For example:

  • Superscript 1: &sup1; → ¹
  • Superscript 2: &sup2; → ²
  • Superscript 3: &sup3; → ³

These entities can be helpful when you need to display superscript characters directly in your HTML.

In summary, the <sup>< element is a powerful tool in HTML for presenting content with superscript, offering flexibility for various use cases. The combination of HTML entities and CSS allows for additional customization to achieve the desired visual effect.

SubScript in HTML

In HTML, a subscript is a style of text formatting that positions characters or numbers below the regular baseline. It is commonly used for representing chemical formulas, mathematical expressions, and other instances where certain elements need to appear lower than the standard text line. In HTML, the element is utilized to denote subscript text.

  1. Chemical Formulas :
    This example represents the chemical formula for water (H₂O). The <sub> element is used to position the "2" below the baseline, indicating subscript.
    2. H<sub>2</sub>O

  2. HTML Entities for Common Subscripts :
    Here, the HTML entity ' <sub> ' is applied to represent the subscript "2" in the chemical formula for carbon dioxide (CO₂).
    3. CO<sub>2</sub>

  3. Mathematical Expressions :
    In mathematical expressions, subscripts are often used to denote different variables or components. Here, "a₁" and "a₂" are represented with the subscript tags.
    4. a<sub>1</sub> + a<sub>2</sub>

Subscript in HTML is a valuable tool for accurately representing scientific and mathematical content on the web. Whether used in chemical formulas or mathematical equations, the element contributes to the clarity and precision of the presented information.
