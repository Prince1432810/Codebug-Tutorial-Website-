Basic structure of HTML


HTML, which stands for HyperText Markup Language, is the standard language used to create web pages. It provides the basic structure for organizing content on the web. Here's a brief explanation of its basic structure:


  1. Document Type Declaration (DOCTYPE) :

    This is the very first line in an HTML document. It tells the web browser which version of HTML the page is written in. For modern HTML documents, it usually looks like <!DOCTYPE html>.

  2. HTML Element :

    The HTML document is enclosed within <html> tags. This element contains all other elements on the page.

  3. Head Section :

    This is where you include meta-information about the HTML document. It contains elements like <head> and <title>. The <title> element sets the title of the webpage, which appears in the browser's title bar or tab.

  4. Body Section :

    This is where the main content of the HTML document is placed. It's enclosed within <body> tags. Content like text, images, videos, links, and more are placed within the body section.

  5. Elements and Tags :

    HTML uses elements and tags to define the structure and content of a webpage. Elements are enclosed within tags, which are represented by angle brackets ('<' and '>'). For example, <p> is the opening tag for a paragraph element, and <∖p> is the closing tag.

  6. Attributes :

    Attributes provide additional information about an element and are placed within the opening tag. They are usually in the form of name-value pairs. For example, the 'href' attribute in an <a> tag specifies the URL of the link.

    7. Here's a very simple example of an HTML document : 
    <!DOCTYPE html>
<html>
<head>
<title>My First Web Page</title>
</head>
<body>
<h1>Welcome to My Website</h1>
<p>This is a paragraph of text.</p>
<img src="image.jpg" alt="An image">
<a href="https://www.example.com">Click here to visit Example.com</a>
</body>
</html>
    In this example :
  • <!DOCTYPE html> declares the document type.
  • <html> encloses the entire HTML document.
  • <head> contains meta-information about the document.
  • <title> sets the title of the webpage.
  • <body> contains the main content of the webpage.
  • <h1>, <p>, <img>, and <a> are elements used to structure and present content within the body section.
  • src and alt are attributes used in the <img> tag.
  • href is an attribute used in the <a> tag.
