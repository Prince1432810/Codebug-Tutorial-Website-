Java Control Flow



Conditional Statements

Conditional statements in Java are used to execute certain blocks of code based on specified conditions. They allow for decision-making within a program, enabling different paths of execution depending on the evaluation of boolean expressions.

  • if Statement :
    The ' if ' statement is the most basic conditional statement in Java. It allows you to execute a block of code if a specified condition is true. 
    • <div class="exp">
 if (condition) {
 // Block of code to be executed if condition is true
}
<div>

  • if-else Statement :
    The ' if-else 'statement extends the functionality of the ' if ' statement by providing an alternative block of code to execute if the condition is false.
    • <div class="exp">
if (condition) {
    // Block of code to be executed if condition is true
} else {
   // Block of code to be executed if condition is false
   }
<div>

  • else-if Ladder :
    The ' else-if ' ladder allows you to test multiple conditions sequentially. If the condition in the ' if ' statement is false, the subsequent ' else-if ' conditions are evaluated until a true condition is found, or the final else block is executed.
    • <div class="exp">
if (condition1) {
  // Block of code to be executed if condition1 is true
} else if (condition2) {
  // Block of code to be executed if condition2 is true
} else {
  // Block of code to be executed if all conditions are false
}
<div>

  • switch Statement :
    The ' switch ' statement evaluates an expression and executes code blocks based on matching cases. It provides a more concise alternative to nested ' if-else ' statements when dealing with multiple conditions.
    •  <div class="exp">
switch (expression) {
case value1:
 // Block of code to be executed if expression equals value1
  break;
case value2:
  // Block of code to be executed if expression equals value2
  break;
default:
  // Block of code to be executed if no cases match
}
<div>

Looping Statements

Looping statements allow you to execute a block of code repeatedly as long as a specified condition is true.

  • for Loop:
    The for loop is used when you know the exact number of iterations required. It consists of three parts: initialization, condition, and iteration expression. 
    <div class="exp">
for (initialization; condition; iteration) {
   // Block of code to be executed
}
<div>

    • Initialization : This part is executed only once before the loop starts. It typically initializes a loop control variable.

    • Condition : This is evaluated before each iteration. If true, the loop continues; if false, the loop terminates.

    • Iteration : This is executed after each iteration of the loop. It typically updates the loop control variable.


  • while Loop:
    The ' while ' loop is used when you want to execute a block of code repeatedly as long as a specified condition is ' true '. 
    <div class="exp">
while (condition) {
  // Block of code to be executed
}
<div>

    • Condition : This is evaluated before each iteration. If 'true ', the loop continues; if ' false ', the loop terminates.

    • Note : Ensure that the condition is eventually ' false ' to prevent infinite loops.

  • do-while Loop:
    The do-while loop is similar to the while loop, but it guarantees that the block of code is executed at least once before the condition is checked. 
    <div class="exp">
do {
  // Block of code to be executed
} while (condition);
<div>

    • Condition : This is evaluated after each iteration. If ' true ', the loop continues; if ' false ', the loop terminates.

Branching Statements

Branching statements alter the normal flow of control in a program.

  • break Statement :
    The ' break ' statement is used to terminate the loop or ' switch ' statement in which it appears. When encountered, it causes control to exit the innermost loop or ' switch ' statement and ' continue ' execution from the statement immediately following the loop or switch.In a ' switch ' statement, ' break ' is used to terminate the switch block. Without a ' break ', execution would continue to subsequent cases until a break or the end of the ' switch ' block is encountered.
    • <div class="exp">
for (int i = 0; i < 10; i++) {
if (i == 5) {
  break; // Exit the loop when i equals 5
}
  System.out.println(i);
}
<div>

  • continue Statement :
    The ' continue ' statement is used to skip the current iteration of a loop and ' continue ' with the next iteration. It is typically used to avoid executing certain statements within a loop under specific conditions.' continue ' is commonly used within loops to skip specific iterations based on certain conditions. After encountering a ' continue ', control immediately jumps to the loop's next iteration.
    • <div class="exp">
for (int i = 0; i < 10; i++) {
if (i % 2 == 0) {
    continue; // Skip even numbers
}
  System.out.println(i);
}
<div>

  • return Statement :
    The ' return ' statement is used to exit from a method, optionally returning a value. It can be used to terminate the execution of a method prematurely, even from within nested control structures. When encountered, ' return ' immediately exits the method in which it appears. If a ' return ' value is specified, the method returns that value to the caller.
    • <div class="exp">
public int calculateSum(int a, int b) {
if (a < 0 || b < 0) {
  return -1; // Return -1 if either parameter is negative
}
  return a + b; // Return the sum of a and b
}
<div>

Understanding and mastering control flow statements is crucial for writing efficient and structured Java programs.
