Input/Output Streams In Java


In Java, input/output (I/O) streams are used to handle reading from and writing to various sources, such as files, network connections, and system input/output. Streams represent sequences of data that can be read from or written to, and they provide a uniform interface for performing I/O operations. Java's I/O system is based on streams, which are categorized into two main types: byte streams and character streams.


  1. Byte Streams :

    • Byte streams (' InputStream ' and ' OutputStream ') are used for reading and writing raw binary data, such as images or binary files.

    • ' InputStream ' and ' OutputStream ' are abstract classes representing input and output streams for bytes, respectively.

    • Example of using ' InputStream ' to read bytes from a file:
      • try (InputStream inputStream = new FileInputStream("example.txt")) {
    int data;
    while ((data = inputStream.read()) != -1) {
        // Process the byte
    }
} catch (IOException e) {
    e.printStackTrace();
}

  2. Character Streams :

    • Character streams (' Reader ' and back' Writer '') are used for reading and writing text data, handling characters and strings.

    • ' Reader ' and ' Writer ' are abstract classes representing input and output streams for characters, respectively.

    • Example of using ' Reader ' to read characters from a file:
      • try (Reader reader = new FileReader("example.txt")) {
    int data;
    while ((data = reader.read()) != -1) {
        // Process the character
    }
} catch (IOException e) {
    e.printStackTrace();
}

  3. Buffered Streams :

    • Buffered streams (' BufferedInputStream ', ' BufferedOutputStream ', ' BufferedReader ', ' BufferedWriter ') provide buffering capabilities, which can improve I/O performance by reducing the number of actual I/O operations.

    • Example of using ' BufferedReader ' to read lines from a file:
      • try (BufferedReader reader = new BufferedReader(new FileReader("example.txt"))) {
    String line;
    while ((line = reader.readLine()) != null) {
        // Process the line
    }
} catch (IOException e) {
    e.printStackTrace();
}

  4. Object Streams :

    • Object streams (' ObjectInputStream ' and ' ObjectOutputStream ' ) are used for reading and writing objects to streams.

    • They allow you to serialize and deserialize objects, converting them into a format that can be stored or transmitted and then reconstructing them back into objects.

    • Example of writing an object to a file:
      • try (ObjectOutputStream oos = new ObjectOutputStream(new FileOutputStream("object.dat"))) {
    MyClass obj = new MyClass();
    oos.writeObject(obj);
} catch (IOException e) {
    e.printStackTrace();
}

These are the basic types of I/O streams in Java, each serving different purposes for reading from and writing to various data sources.
