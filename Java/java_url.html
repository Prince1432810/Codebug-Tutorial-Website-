URL in Java:


The URL class in Java is used to represent a Uniform Resource Locator, which is a pointer to a resource on the internet. It provides a convenient way to parse, construct, and manipulate URLs. Here's a detailed description of the URL class:


  1. Creating a URL object :
    You can create a URL object using one of its constructors, which takes a string representation of the URL as an argument. For example:
    2. URL url = new URL("https://www.example.com");

  2. Accessing URL components :
    The URL class provides methods to access different components of the URL, such as the protocol, host, port, path, query, and fragment. For example:
    3. String protocol = url.getProtocol(); // returns "https"
  String host = url.getHost(); // returns "www.example.com"
  int port = url.getPort(); // returns -1 if not specified in the URL
  String path = url.getPath(); // returns "/"
  String query = url.getQuery(); // returns null if not specified
  String fragment = url.getFragment(); // returns null if not specified

  3. Building URLs :
    You can construct URLs by combining different components using the URL constructor or the URL class's toExternalForm() method. For example:
    4. URL url = new URL("https", "www.example.com", "/path", "param=value");
  String urlString = url.toExternalForm(); // returns "https://www.example.com/path?param=value"

  4. URL encoding :
    The URL class provides methods for encoding and decoding URLs to ensure that special characters are properly handled. For example:
    5. String encodedPath = URLEncoder.encode("/path with spaces", StandardCharsets.UTF_8);


In Java, the URL class represents a Uniform Resource Locator, which is a pointer to a resource on the internet. It provides methods for accessing the various components of the URL, such as the protocol, host, port, path, etc. Here's a brief overview of the URL class and its usage:

import java.net.URL;

  public class Main {
      public static void main(String[] args) {
          try {
              // Create a URL object
              URL url = new URL("https://www.example.com");
  
              // Get the protocol (e.g., http, https)
              System.out.println("Protocol: " + url.getProtocol());
  
              // Get the host name
              System.out.println("Host Name: " + url.getHost());
  
              // Get the port number
              System.out.println("Port Number: " + url.getPort());
  
              // Get the path
              System.out.println("Path: " + url.getPath());
  
              // Get the query string
              System.out.println("Query String: " + url.getQuery());
  
              // Get the file name
              System.out.println("File Name: " + url.getFile());
  
              // Get the full URL as a string
              System.out.println("Full URL: " + url.toString());
          } catch (Exception e) {
              e.printStackTrace();
          }
      }
  }

HttpURLConnection In Java


The HttpURLConnection class in Java is a subclass of URLConnection that provides support for HTTP-specific features. It allows you to establish connections to HTTP servers, send HTTP requests, and read HTTP responses. Here's a detailed description of the HttpURLConnection class:

  1. Creating a connection :
    You can create an HttpURLConnection object by calling the openConnection() method on a URL object and casting the result. For example:
    2. URL url = new URL("https://www.example.com");
  HttpURLConnection conn = (HttpURLConnection) url.openConnection();

  2. Configuring the request :
    Before sending a request, you can configure properties such as the request method, headers, and timeouts. For example:
    3. conn.setRequestMethod("GET");
  conn.setRequestProperty("User-Agent", "Mozilla/5.0");
  conn.setConnectTimeout(5000);
  conn.setReadTimeout(5000);

  3. Sending a request :
    To send a request, you can call the getInputStream() method to get the response body as an input stream. For example:
    4. InputStream inputStream = conn.getInputStream();

  4. Handling the response :
    After sending the request, you can read the response from the input stream and process it as needed. For example:
    5. BufferedReader reader = new BufferedReader(new InputStreamReader(inputStream));
  String line;
  StringBuilder response = new StringBuilder();
  while ((line = reader.readLine()) != null) {
      response.append(line);
  }
  reader.close();

  5. Closing the connection :
    It's important to close the connection when you're done with it to release resources. For example:
    6. conn.disconnect();


The HttpURLConnection class allows you to perform HTTP-specific operations, such as sending and receiving data over HTTP. Here's an example of how you can use HttpURLConnection to make an HTTP GET request and read the response:

import java.io.BufferedReader;
  import java.io.InputStreamReader;
  import java.net.HttpURLConnection;
  import java.net.URL;
  
  public class Main {
      public static void main(String[] args) {
          try {
              // Create a URL object
              URL url = new URL("https://www.example.com");
  
              // Open a connection to the URL
              HttpURLConnection conn = (HttpURLConnection) url.openConnection();
  
              // Set the request method
              conn.setRequestMethod("GET");
  
              // Read the response
              BufferedReader reader = new BufferedReader(new InputStreamReader(conn.getInputStream()));
              String line;
              StringBuilder response = new StringBuilder();
              while ((line = reader.readLine()) != null) {
                  response.append(line);
              }
              reader.close();
  
              // Print the response
              System.out.println(response.toString());
  
              // Disconnect the connection
              conn.disconnect();
          } catch (Exception e) {
              e.printStackTrace();
          }
      }
  }
