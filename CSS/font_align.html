Font Alignment

Font alignment, also known as text alignment, refers to the horizontal alignment of text within a container such as a paragraph, a heading, or a table cell. There are typically four types of text alignment :

  1. Left-aligned :

    Text is aligned along the left edge of the container. This is the default alignment for most languages that are written from left to right.

  2. Right-aligned :

    Text is aligned along the right edge of the container. This is commonly used for numbers or languages that are written from right to left, such as Arabic or Hebrew.

  3. Center-aligned

    Text is aligned along the center of the container. This creates a visually balanced appearance and is often used for headings or titles.

  4. Justified :

    Text is aligned along both the left and right edges of the container, with extra spacing between words as needed to fill the entire width. This creates a clean, uniform look but can sometimes result in awkward spacing if not implemented carefully.

    5. In HTML/CSS, you can control text alignment using the text-align property:

    .left-align {
    text-align: left;
}

.right-align {
    text-align: right;
}

.center-align {
    text-align: center;
}

.justify {
    text-align: justify;
}

Text Decoration :

Text decoration refers to the visual enhancements applied to text to improve readability or add emphasis. Common text decorations include:

  1. Underline :
    Adding a line underneath the text.
  2. Overline :
    Adding a line above the text.
  3. Line-through :
    Drawing a line through the middle of the text.
  4. Blink :
    Making the text blink on and off, though this is now rarely used due to accessibility concerns and browser support.

In HTML/CSS, you can apply text decoration using the text-decoration property:

.underline {
    text-decoration: underline;
}

.overline {
    text-decoration: overline;
}

.line-through {
    text-decoration: line-through;
}

.blink {
    text-decoration: blink;
}

These properties can be applied to individual elements or combined as needed to achieve the desired visual effect. Keep in mind that some text decorations may not be widely supported or may have accessibility implications, so it's essential to use them judiciously.
