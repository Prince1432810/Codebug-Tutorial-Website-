Line Height

In CSS (Cascading Style Sheets), line height and letter spacing are properties that can be applied to text elements to control the spacing between lines and characters respectively. Let's delve into each property :

The line-height property in CSS defines the amount of vertical space between lines of text within an element. It can be specified using various units such as pixels, ems, percentages, or even unitless values.

Syntax : 

selector {
    line-height: value;
}

Values :

  1. Normal :
    This value sets the line height to a value that is normal for the current font. It is typically around 1.2 to 1.4 times the font size.
  2. Length or Percentage :
    You can specify an absolute length (e.g., px, em, rem) or a percentage of the font size to set a specific line height.
  3. Unitless Number :
    You can use a unitless number to multiply the font size, setting the line height proportional to the font size.
  4. Initial/Inherit :
    These values set the line height to its default value or inherit from the parent element, respectively.

    5. Example : 

    p {
    line-height: 1.5; /* Unitless value, line height is 1.5 times the font size */
}

Letter Spacing in CSS :

This value resets any previous letter-spacing adjustment, typically resulting in the default spacing for the font.

Syntax : 

selector {
    letter-spacing: value;
}

Values :
  1. Normal :
    This value resets any previous letter-spacing adjustment, typically resulting in the default spacing for the font.
  2. Length :
    You can specify an absolute length (e.g., px, em, rem) to set a specific amount of spacing between characters.
