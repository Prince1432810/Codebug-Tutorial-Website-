Units in css

In CSS (Cascading Style Sheets), units are used to specify the measurement of various properties such as length, width, margin, padding, font size, etc. Units in CSS provide flexibility and control over the layout and appearance of web pages. There are different types of units in CSS, each serving a specific purpose. Let's explore them in detail :

  1. Absolute Length Units :

    These units are fixed and do not change with the size of the viewport or the device. They include :

    • Pixels(px) :
      Pixel is a relative unit, representing one device pixel on the screen. It's commonly used for specifying precise dimensions or alignment.
    • Points (pt) :
      Point is another absolute unit often used in print media. One point is equal to 1/72 inch. In CSS, 1 point is approximately equal to 1.33 pixels.
    • Inches(in), Centimeters(cm), Melimeters(mm) :
      These units represent physical lengths and are mostly used for print styling.

  2. Relative Length Units :

    Relative units are defined in relation to other elements or the viewport. They provide scalability and responsiveness across different devices. Common relative units include :

    • Percentage(%) :
      Percentage units are relative to the parent element. For example, setting a width of 50% on a div element makes it half the width of its containing block.
    • Em(em) :
      One em unit is equal to the computed font-size of the element itself. For instance, if the font-size of an element is 16px, 1em is equal to 16px.
    • Rem(rem) :
      Similar to em, but rem units are relative to the font size of the root element (html). This makes rem units more predictabel and easier to manage in complex layouts.

  3. Font-relative Length Units :

    These units are specifically used to define text-related properties such as font size, line height, etc. They include:

    • Absolute-size Keywords :
      These keywords represent predefined font sizes, with 'medium' being the default size.
    • Relative-size Keywords(smaller, larger) :
      These keywords adjust the font size relative to the parent element's font size.
    • Ch (Character Units) :
      One ch unit represents the width of the '0' (zero) character in the current font.

  4. Other Units :

    Some less commonly used units include :

    • Ex (X-height) :
      One ex unit represents the x-height of the current font, typically the height of the lowercase letter 'x'.
    • Grid Units (fr) :
      Used in CSS Grid Layout, fr unit represents a fraction of the available space in the grid container.
    • Angle Units (deg, rad, grad, turn) :
      These units are used to specify angles for CSS properties like transform and gradient directions.

    5. Examples :

    /* Absolute Length Units */
div {
  width: 200px;
  margin: 1in;
  padding: 2cm;
}

/* Relative Length Units */
div {
  width: 50%;
  font-size: 1.2em;
  margin: 2rem;
}

/* Viewport Percentage Lengths */
div {
  width: 50vw;
  height: 75vh;
}

/* Font-relative Length Units */
div {
  font-size: medium;
  line-height: larger;
}

/* Other Units */
div {
  width: 20ch;
  grid-template-columns: 1fr 2fr;
  transform: rotate(45deg);
}

Understanding and mastering CSS units allows web developers to create flexible and responsive layouts that adapt well to various screen sizes and devices. Choosing the appropriate units for different properties is crucial for achieving the desired design and user experience.
