FlexBox

display: flex is a CSS property that enables a flex container to control the layout of its children elements. It's a part of the Flexbox layout module, which provides a more efficient way to arrange, align, and distribute space among items in a container, even when their size is unknown or dynamic.

Here's a detailed explanation of display: flex and its various properties:

Basics of FlexBox :

  1. Flex Container :
    • When you apply display: flex to an element, it becomes a flex container, and its direct children become flex items.
    • The flex container establishes a new flex formatting context for its children.

  2. Flex Items :
    • These are the direct children of the flex container.
    • They can be laid out in any direction: horizontally (in a row) or vertically (in a column).
    • By default, flex items will try to shrink or grow to fill the available space.

Properties of Flex Container :

  1. flex-direction :
    • Specifies the direction of the main axis along which flex items are laid out.
    • Values: row (default), row-reverse, column, column-reverse.

  2. flex-wrap :
    • Controls whether the flex container is single-line or multi-line, and the direction in which new lines are stacked.
    • Values: nowrap (default), wrap, wrap-reverse.

  3. justify-content :
    • Aligns flex items along the main axis of the flex container.
    • Values: flex-start (default), flex-end, center, space-between, space-around, space-evenly.

  4. align-items :
    • Aligns flex items along the cross axis of the flex container.
    • Values: stretch (default), flex-start, flex-end, center, baseline.

  5. align-content :
    • Aligns flex lines (if there are multiple lines) within the flex container when there is extra space on the cross-axis.
    • Values: stretch (default), flex-start, flex-end, center, space-between, space-around.

Properties of Flex Items :

  1. order :
    • Specifies the order in which flex items appear within their flex container.
    • Default value: 0.

  2. fle-grow :
    • Determines how much a flex item should grow relative to the rest of the flex items when there is extra space along the main axis.
    • Default value: 0.

  3. flex-shrink :
    • Determines how much a flex item should shrink relative to the rest of the flex items when there is not enough space along the main axis.
    • Default value: 1.

  4. flex-basis :
    • Specifies the initial main size of the flex item before any remaining space is distributed according to the flex-grow and flex-shrink properties.
    • Default value: auto.

  5. flex :
    • Shorthand property for flex-grow, flex-shrink, and flex-basis.
    • Syntax: <flex-grow> <flex-shrink> <flex-basis>.
    • Default value: 0, 1, auto.

Example Usage : 

.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
}

.item {
  flex: 1; /* This will make each item grow equally to fill the container */
}


<div class="container">
  <div class="item">>tem 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>

This setup will create a flex container with three items evenly spaced along the main axis. The items will also be vertically centered because of the align-items: center property. The flex: 1 property ensures that each item takes up an equal amount of available space within the container.
