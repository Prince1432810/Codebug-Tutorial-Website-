Text Align Property

The text-align property in CSS is used to specify how inline content like text is horizontally aligned within its containing element. It is a very useful property for controlling the alignment of text within elements like paragraphs, headings, lists, etc. Here's a detailed breakdown of the text-align property:

Syntax : 

selector {
    text-align: value;
}

Values :

  1. left :
    Aligns text to the left edge of the containing element. This is the default value.
    2. text-align: left;

  2. right :
    Aligns text to the right edge of the containing element.
    3. text-align: right;

  3. center :
    Centers the text within the containing element.
    4. text-align: center;

  4. justify :
    Stretches the lines of text to fill the width of the container, except for the last line.
    5. text-align: justify;

Example Usage : 

/* Align paragraphs to the center */
p {
    text-align: center;
}

/* Align headings to the right */
h1, h2, h3 {
    text-align: right;
}

/* Justify text within a div */
.container {
    text-align: justify;
}

Notes :

  • The text-align property applies to inline-level and inline-block elements, and it will affect the alignment of their inline content.

  • It doesn't affect block-level elements directly, but it will affect inline content inside block-level elements.

  • When applying text-align: justify;, browsers will adjust the spacing between words to create a straight left and right edge for the text block, except for the last line, which remains aligned to the left.

  • It's worth noting that text-align doesn't affect vertical alignment. For vertical alignment, you might use other properties like vertical-align or flexbox/grid alignment properties.

Browser Compatibility :

The text-align property is widely supported by modern browsers and is part of the CSS1 specification, making it compatible with most web development projects. However, it's always a good practice to check for compatibility if you have specific browser requirements.
